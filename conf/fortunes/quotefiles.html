Quote Files

These files contain quotes from TV shows. They were created for use with GUIquotes but will work with any program which supports UNIX fortune format files.

For random examples of quotes from some of these files see the Episode Guide section of this website.

To use the files with GUIquotes, first unzip them and move the text file to the quotes_program\quotes folder (directory). Then run GUIquotes and use the menu selections File -> Change Quote Files to add them to the Selected Quote Files.
